# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-16

### Added
- Free-hand pen drawing with adjustable brush size and colors
- Eraser tool with proportional size
- Shape tools: Line, Rectangle, Ellipse, Arrow
- Text insertion tool
- Spotlight mode for presentations
- Whiteboard mode (white/black backgrounds)
- Multi-monitor support with screen selector
- Minimized UI mode (vertical toolbar)
- Undo/Redo system (up to 20 states)
- Screenshot capture and save
- Multi-language support (English / Spanish)
- Keyboard shortcuts for all tools
- Fullscreen transparent overlay (X11 and Wayland via XWayland)

### Fixed
- Canvas now properly resizes to match each monitor's resolution when switching screens
